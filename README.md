Webprogramozás I. Gyakorlati beadandó
=====================================

Ez a repo a Neumann János Egyetem (NJE) **Webprogramozás I.** kurzusának gyakorlati beadandó feladatához készült, amit a **Blade** csapat készített.

Áttekintés
==========

A feladat a https://vaszilijedc.hu weboldal átdolgozása modernebb elgondolásban. A projekt megvalósítása során a cél egy letisztultabb, reszponzívabb és funkcionálisabb felület létrehozása volt az eredeti oldal alapjaira építve.

Éles demo
=========

A projekt működés közben megtekinthető az alábbi linken:
https://vaszilij.biglan.net

Felhasznált technológiák és nyelvek
===================================

A fejlesztés során az alábbi nyelveket és technológiákat alkalmaztuk:

* HTML5
* CSS3
* JavaScript
* PHP

Külső könyvtárak és eszközök:

* Bootstrap (a reszponzív szerkezetért)
* jQuery (a kliensoldali szkriptekhez)
* Summernote (szövegszerkesztő interfész)
